GodotWE
================================================================================
A recreation of SMM:WE 3.5.1 in Godot Engine.

Currently, GodotWE is in a very early stage and only mostly features an editor
with playtesting functionality. We have not released any builds ready to play,
however you can build it yourself.

 > [!NOTE]
 > Want to see news and content about GodotWE? Join our
 > [Discord server](https://discord.gg/xX9MCyyJ4N)!

Getting started
--------------------------------------------------------------------------------
To use GodotWE's source code, you will need:

 - The latest Godot Engine version (currently 4.6.2.stable)
 - Git (optional but recommended)

To start working with GodotWE, follow these steps:

1. Download the main branch. You can do this either from the `Code` button on
   the GitHub website or by cloning into the repository using Git:
   ```
   git clone https://github.com/NCubss/GodotWE.git
   ```
2. Open the Godot editor and `📂 Import` the `project.godot` file.
3. Enjoy! Modify the game to your heart's content.
